## 📖 Mini Guia de Estudo

### Estruturas de Repetição

* **for** → usado quando o número de repetições é conhecido
* **while** → usado quando a condição é verificada antes
* **do-while** → executa pelo menos uma vez

### Exemplo em C:

```c
for(int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
```

---

## 📘 Glossário

* **Algoritmo**: sequência de passos para resolver um problema
* **Variável**: espaço na memória para armazenar dados
* **Loop**: estrutura de repetição
* **Fluxograma**: representação gráfica de um algoritmo
* **Pseudocódigo**: descrição textual de um algoritmo
